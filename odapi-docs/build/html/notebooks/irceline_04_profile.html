Time Series Profiles¶

Finding profiles in Time Series is a common task to perform. If time series are impacted by human activity it is generally a good idea to investigate if there is a weekly pattern on it.

Time Series toolbox provides time grouper for groupby method and for this case the weekly grouper which maps all records to a hypothetic week located at the beginning of the year 1940. This hypthetic week is very convenient as it conserves day of week and order it from Monday 1st to Sunday 7th.

Imports¶

We import required packages:

[1]

import pandas as pd
import matplotlib.pyplot as plt
import matplotlib.dates as mdates
from odapi.connectors import Irceline

[2]

from odapi.settings import settings
settings.logger.setLevel(40)

Dataset & Client¶

We load the dataset we previously stored and create the Irceline client:

[3]

data = pd.read_pickle("dump/brussels_2014.pickle")
client = Irceline()

Week profiles¶

To use time grouper, just reference the desired method to groupby and then aggregate:

[4]

p = data.groupby(client.weekly).mean()

Graphically, it renders as follow:

[6]

scales = client.scales()
for k in ['NO', 'NO2', 'O3']:#p.columns.levels[2]:
    axe = p.filter(regex='{}/'.format(k)).droplevel([1,2,3], axis=1).plot()
    axe.set_title("Week Profile 2014")
    axe.set_xlabel("Time, $t$")
    axe.set_ylabel("Series, $x(t)$")
    axe.set_ylim(scales[k])
    lgd = axe.legend(bbox_to_anchor=(1, 1), loc='upper left')
