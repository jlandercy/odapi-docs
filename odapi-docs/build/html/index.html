Open Data API¶

Welcome to Open Data API project documentation page.

Open Data API (abbreviated to odapi) is a Python3 package providing a generic interfaces on top of multiple data sources and data models in order to abstract their differences and manipulate them in a standardized way.

Open Data API mainly standardizes Time Series and Geomatic management using the powerful Pandas DataFrame object under the hood.

Open Data API also defines flexible interfaces to control how data model and flow are built and enforce quality and consistency of connectors developments.

Installation¶

Open Data API uses classic setuptools flow, to install the package just follow the usual procedure:

python3 setup.py install

The package repository is available at https://github.com/jlandercy/odapi.

A KISS interface¶

We aim to fit with the KISS principle. Download time series data with Open Data API package is as easy as:

1
2
3
4
5
from odapi.connectors import Irceline

client = Irceline()
meta = client.select(sitekey='41', measurekey=['NO'])
recs = client.get_records(meta)

This light-weight snippet above download the last seven days of Nitrogen Oxides for Brussels from the Irceline Air Quality SOS API.
